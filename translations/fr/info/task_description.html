On vous donne une liste de nombres entiers positifs et un nombre entier positif N. Vous devez trouver la N-ième puissance de l'élément d'indice N de la liste. Si N est supérieur au plus grand indice de la liste, alors vous devez retourner -1. N'oubliez pas que le premier élément d'une liste se trouve à l'indice 0.

Regardons quelques exemples :
- liste = [1, 2, 3, 4] et N = 2, alors le résultat est 32 == 9;
- liste = [1, 2, 3] et N = 3, mais l'indice N est supérieur au plus grand indice de la liste, donc le résultat est -1.

Entrée : Deux arguments. Une liste (list) d'entiers (int) et un indice en tant qu'entier (int).

Sortie : Le résultat en tant qu'entier (int).

Exemple : 

index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1

À quoi cela sert : Cette mission vous apprend à utiliser les listes et les indices, ainsi que des mathématiques simples.

Précondition : 0 < len(liste) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in liste)
