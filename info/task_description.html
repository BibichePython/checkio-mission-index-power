You are given an array with positive numbers and a number N. You should find find N-th power of N-the element of the array. If N is outside on the array, then return -1. Don't forget that the first element has 0 index.

Let's look at examples:
- array = [1, 2, 3, 4] and N = 2, then the result is 32 == 9;
- array = [1, 2, 3] and N = 3, but N is greater than the length of the array, so the result is -1.

Input: Two arguments. An array as a tuple of integers and a number as a integer.

Output: The result as an integer.
